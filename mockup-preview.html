Cette maquette visuelle montre l’identité de l’application avant toute implémentation Angular. Le ton choisi est enterprise-premium : une interface métier propre, sérieuse, mais avec un vrai parti pris typographique et spatial.

Visual Notes

People Directory should not look like a default Angular Material starter.

Palette minérale, surfaces papier, accent profond, hiérarchie éditoriale, et composants métier qui restent sobres. L’idée n’est pas d’avoir une app “fun”, mais une app clairement conçue.

App Shell + List Page

La liste est la vue la plus opérationnelle. Elle doit être dense sans être froide, très claire sans ressembler à un simple tableau administratif.
